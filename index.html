Setup

1. Make a new Supabase project

Create a new project in your Supabase dashboard to get started.

2. Environment

Create a .env.local file and add the following values:

SUPABASE_URL=EXAMPLE SUPABASE_KEY=EXAMPLE

3. Create new tables (using supabase SQL editor)

Categories:

create table public."Categories" ( slug text not null, name text not null, img text null, created_at timestamp with time zone not null default now(), is_featured boolean null default false, constraint Categories_pkey primary key (slug) ) TABLESPACE pg_default;

Products:

create table public."Products" ( slug text not null, name text not null, price real not null, category text not null, description text not null, discount smallint not null default '0'::smallint, quantity smallint not null default '0'::smallint, quality real not null, created_at timestamp with time zone not null default now(), featured_promotion boolean not null default false, top_selling boolean not null default false, images text[] not null default '{}'::text[], related_products text[] not null default '{}'::text[], constraint Products_pkey primary key (slug), constraint Products_category_fkey foreign KEY (category) references "Categories" (slug), constraint Products_discount_check check ((discount <= 100)), constraint Products_quality_check check ( ( (quality>= (0)::double precision) and (quality <= (5)::double precision) ) ) ) TABLESPACE pg_default;

Config:

create table public."Config" ( slug text not null, "themeStringObj" text null default '{"primary":"blue","secondary":"indigo"}'::text, tos text null default ''::text, about_us text null default ''::text, mission text null default ''::text, partners_description text null default ''::text, connect_description text null default ''::text, constraint Config_pkey primary key (slug) ) TABLESPACE pg_default;

And create 1 row with the slug of "general", leave other columns to default.

Sets:

create table public."Sets" ( slug text not null, name text not null, images text[] not null default '{}'::text[], made_by text not null, components text[] not null default '{}'::text[], description text not null, tags text[] not null default '{}'::text[], created_at timestamp with time zone null default now(), related_products text[] not null default '{}'::text[], price real not null default '0'::real, discount real not null default '0'::real, constraint Sets_pkey primary key (slug) ) TABLESPACE pg_default;

Team:

create table public."Team" ( slug text not null, name text not null, role text null, img text null, constraint Team_pkey primary key (slug) ) TABLESPACE pg_default;

Partners:

create table public."Partners" ( slug text not null, img text not null, constraint Partners_pkey primary key (slug) ) TABLESPACE pg_default;

Themes:

create table public."Themes" ( slug text not null default 'general'::text, "themeStringObj" text not null default '{"primary":"blue","secondary":"violet"}'::text, img text null default ''::text, constraint Themes_pkey primary key (slug) ) TABLESPACE pg_default;

Orders:

create table public."Orders" ( slug text not null, created_at timestamp with time zone null default now(), name text not null, email text not null, address text not null, city text not null, postal_code text null, notes text null, payment_method text not null, ordered_products text not null, shipping_fee real not null, sub_total real not null, phone integer not null, constraint Orders_pkey primary key (slug) ) TABLESPACE pg_default;

4. Create new storage buckets

Create these storage buckets (all public):

  • categories-images
  • products-images
  • sets-images
  • team-images
  • partners-images
  • themes-images

5. Create Admin account

Go to Supabase authentication and create a new account manually (email & pass). Turn off the option "Allow new users to sign up", and you can get the uuid from the users table.

6. SQL Policies (using supabase SQL editor)

Go to ./setup-files/create-sql-polices.js and write the table name and admin uuid (you must do it for each table), using node create-sql-polices.js it will generate an SQL Policies code file for the Table or Bucket, (you'll need to paste the results of that file in Supabase SQL editor).

7. Constants

Go to ./src/helpers/config.js and edit whatever constant you need there.

8. Language

Go to ./src/helpers/language.js and edit any string you want in the website (e.g change "Sets" to "Builds" or re-write everything in another language), and you can also change the direction of the website if you want to use Arabic or Hebrew language.

9. Logo and banner

Go to ./public/ and change the icon.png

Go to ./src/app/assets and change the header-bg.png (optional)

10. Manage tables

Go to /admin and log in with your admin account.

You will be navigated to /admin/panel where you can easily manage categories, products, sets, team, partners, general info and themes.
