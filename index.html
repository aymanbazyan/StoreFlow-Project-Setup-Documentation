Setup

1. Setup PostgreSQL (Ubuntu)

1. Download PostgreSQL to get started.

sudo apt update
sudo apt install postgresql postgresql-contrib sudo service postgresql start

2. Open the psql shell.

sudo -u postgres psql

3. Create your database.

CREATE DATABASE 'your_database_name';

4. Create admin account.

CREATE USER 'your_username' WITH ENCRYPTED PASSWORD 'your_password';

5. Give the account admin privileges.

GRANT ALL PRIVILEGES ON DATABASE 'your_database_name' TO 'your_username';
ALTER USER 'your_username' CREATEDB;

After these commands, type \q or exit to leave the psql prompt.

6. Connect to your_database_name as the postgres superuser. It's a good way to verify the database exists.

sudo -u postgres psql -d 'your_database_name'

2. Environment

Create a .env.local file then add & fill the following values:

# Genreal
NODE_ENV='development'
APP_BASE_URL="http://localhost:3000"

# Auth
JWT_SECRET="your-very-strong-jwt-secret-key-here-at-least-32-characters"

# Database
JWT_SECRET="postgresql://username:password@localhost:5432/your_database_name"

# Email
SERVICE=gmail
EMAIL_MAIL=example@gmail.com
EMAIL_PASS=fdoa kgsa bmrw asnt

# Checkout
CHECKOUT_LIMIT=3 # per reset time
CHECKOUT_LIMIT_RESET_TIME=3600000 # 1 hour

3. Connect the app to the database and initialize the schemas

npm run prisma:init
npm run prisma:migrate:dev
npm run prisma:generate
npm run prisma:push

4. Create Admin account

Open the website with npm run dev

Go to /account and create a new account manually (name, email, pass).

Open the prisma studio with npm run prisma:studio

Go to the Users table

Manually change isAdmin to true

5. Constants

Go to ./src/helpers/config.js and edit whatever constant you need there.

6. Language

Go to ./src/helpers/language.js and edit any string you want in the website (e.g change "Sets" to "Builds" or re-write everything in another language), and you can also change the direction of the website if you want to use Arabic or Hebrew language.

7. Logo and banner

Go to ./public/ and change the icon.png

Go to ./src/app/assets and change the default header-bg.png (optional), (you can change it in admin panel now)

8. Manage tables

Go to /account and log in with your admin account (admin account from step 4, not 1).

You will find in account dropdown a button to navigate to /admin where you can easily manage categories, products, sets, team, partners, general info and themes.
